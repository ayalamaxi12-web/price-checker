📋 Planillas de PMs adicionales

⚙ Reglas de Precio

Editá el porcentaje adicional y el costo de envío de cada regla. El envío se suma cuando el precio base ≥ al umbral configurado.

🛠 Configuración General

📦 Envío Gratis

Umbral de envío gratis
Precio mínimo en ML a partir del cual el sistema suma el costo de envío
Costo de envío por defecto
Monto a sumar cuando no se conoce el último costo real

🎯 Tolerancias de Discrepancia

Tolerancia verde (OK)
Diferencia máxima en % para considerar el precio correcto
Tolerancia amarilla (Advertencia)
Diferencia máxima en % antes de marcar como error crítico
